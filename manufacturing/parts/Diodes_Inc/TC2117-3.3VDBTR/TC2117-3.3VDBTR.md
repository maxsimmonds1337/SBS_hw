# TC2117-3.3VDBTR — 800mA Fixed 3.3V LDO (SOT-223)

## Key Specs

| Parameter | Value |
|-----------|-------|
| Vin range | 2.7V to 6.5V |
| Vout (fixed) | 3.3V |
| Iout max | 800 mA |
| Dropout voltage | 500 mV @ Iout=800mA |
| Quiescent current | 1 mA (typ) |
| PSRR | 65 dB @ 1 kHz |
| Package | SOT-223 (DBTR = tape & reel) |

## Input Rail: +5V from DRV8301 Buck

```
Vin = +5V (DRV8301 internal buck)
Vout = 3.3V

Headroom = 5.0 - 3.3 = 1.7V
Dropout at max load = 500mV
Available headroom vs dropout: 1.7V > 0.5V  ✓

Maximum load current = 800mA
```

## Load Estimate

Devices on +3.3V rail:
| Device | Current (typ) |
|--------|-------------|
| STM32F405 (active, 168MHz) | ~80 mA |
| NRF51822 EYSGJNZWY (Tx) | ~15 mA |
| BMI160 | ~1 mA |
| TCAN1051VDQ1 (VIO) | ~2 mA |
| Pull-ups, logic | ~10 mA |
| **Total** | **~108 mA** |

Well within 800mA limit. Thermal dissipation at 108mA:
```
P = (Vin - Vout) × Iout = (5.0 - 3.3) × 0.108A = 0.184W
Tj = Ta + P × θja = 40 + 0.184 × 85 = 55.6°C  ✓  (SOT-223 θja ≈ 85°C/W)
```

## Decoupling

Per datasheet:
- Input: 1µF min (100nF + 2.2µF in schematic ✓)
- Output: 1µF min (100nF in schematic — consider adding 2.2µF for transient response)
