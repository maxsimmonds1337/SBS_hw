## Why DSS210 Replaces PMEG6020ER

| | PMEG6020ER (current) | DSS210 (replacement) |
|--|--|--|
| Vr max | **20 V** | **100 V** |
| Bus voltage (14S) | 58.8 V — **2.94× over-voltage** | 58.8 V — 1.70× margin |
| Package | SOD-323 (smaller) | SOD-123FL |
| LCSC | — | C511868 |

The bootstrap diode must block the full bus voltage (Vbus = 58.8V) when the high-side MOSFET is OFF. PMEG6020ER at 20V will fail immediately at 14S.

## Bootstrap Circuit Operation

```
DRV8301 bootstrap charging sequence per phase (e.g., Phase A):
  1. Low-side FET turns ON → SH_A node pulled to GND (~0V)
  2. +5V (VBOOST from DRV8301) forward-biases the bootstrap diode
  3. Charge flows: +5V → diode → BST_A cap → SH_A (GND)
  4. BST_A cap charges to ~4.5V (5V - Vf_diode)
  5. Low-side FET turns OFF; SH_A rises to Vbus (58.8V)
  6. BST_A cap is now floating at (58.8 + 4.5) = 63.3V above GND
  7. Diode must block (63.3 - 5) = 58.3V in reverse → requires Vr > 58.8V
```

The VBOOST supply in DRV8301 is ~5V (internal buck output). Reverse voltage
on the diode = Vbus - Vboost ≈ 58.8 - 5 = 53.8V. With 10% margin: 53.8 × 1.1 = 59.2V.
DSS210 at Vr=100V provides 1.69× safety factor.

## Current Rating Check

Bootstrap cap charge current is a short pulse (< 1 µs) during the dead-time window:

```
BST cap size (DRV8301 sheet): 220 nF per phase
Peak charge current ≈ (Vboost - Vf) / Z_charge_loop
  Z_charge_loop ≈ 1-5 Ω (PCB trace resistance + diode bulk resistance)
  I_peak = 4.5V / 2Ω = 2.25 A  (worst case, very short pulse)

Average charge current per switching cycle:
  I_avg = Qg_HS / t_cycle = 93 nC / 50 µs = 1.86 mA

DSS210 If = 2A continuous — peak pulse well within rating  ✓
```

## Placement (3 diodes required)

The VESC 6 MK5 and this design both need one bootstrap diode per phase:
- **BST_A**: between VBOOST (+5V) and BST_A pin of DRV8301 — currently PMEG6020ER ✓ but wrong voltage
- **BST_B**: between VBOOST and BST_B pin — **MISSING in current schematic**
- **BST_C**: between VBOOST and BST_C pin — **MISSING in current schematic**

All three must be added/replaced with DSS210 in DRV8301.kicad_sch.
